Daily Tasks 埋め込みサンプル

使い方

  1. Daily Tasksアプリでデータを入力
  2. 入力データをJSON形式で取得
  3. JSONをBase64エンコード
  4. iframeのURLパラメータとして渡す

サンプルデータの埋め込み

埋め込みコード例

<iframe 
    src="http://localhost:3000/embed.html?data=YOUR_BASE64_ENCODED_DATA"
    width="100%"
    height="400"
    frameborder="0">
</iframe>

JavaScriptでの実装例

// タスクデータ
const taskData = {
    "睡眠時間": { "時間": 7, "分": 0 },
    "起床": { "時": 7, "分": 30 },
    "散歩": { "実施": true, "ゴミ拾い": true },
    "朝食の栄養カバレッジ": "3",
    "体操": true,
    "労働": { "状態": "した", "passion": 80, "discipline": 90, "備考": "" },
    "ジム": "有酸素+筋トレ",
    "勉強会": true,
    "個人開発": true,
    "あすけん": 85,
    "体重測定": true
};

// Base64エンコード（日本語対応）
const encodedData = btoa(encodeURIComponent(JSON.stringify(taskData)));

// iframeのsrc属性を設定
const iframe = document.getElementById('daily-tasks-iframe');
iframe.src = `embed.html?data=${encodedData}`;
